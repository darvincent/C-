这是学习C# 写的第一个程序，目的是为了提升技术支持部的工作效率， 是一个C/S 架构的程序， SLS是server， supportLogSheet是客户端。

主要功能：
1.存储客户信息，内部员工信息，供快速查询
2.提供权限系统，将员工分为普通员工和管理员两个层级。管理员拥有所有权限，普通员工可按需由管理员授予相应权限
3.存储客户系统问题集，并与客户信息相关联。管理员负责问题分配给普通员工处理，普通员工负责更新问题处理信息
4.提供To-do功能，员工可给待办事项设置计划。到指定时间，通知系统会给予提醒
5.提供对历史数据的统计分析功能

运行平台： windows
相关技术： C# .net + sql server
